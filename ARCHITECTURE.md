## ğŸ› ï¸ Technology Details

### Frontend (Browser)
- **Runtime**: V8 JavaScript Engine
- **Storage**: localStorage API
- **Threading**: Single-threaded event loop
- **Network**: Fetch API / CORS
- **Offline**: Service Worker / Cache API

### Backend (Python)
- **Framework**: Flask (micro-framework)
- **ORM**: SQLAlchemy (object-relational mapping)
- **Auth**: Flask-JWT-Extended
- **Server**: Development: Werkzeug, Production: Gunicorn
- **Database**: SQLite (dev), PostgreSQL (prod)

### Database (SQLAlchemy)
- **Type**: Relational (SQL)
- **Relationships**: One-to-Many, One-to-One
- **Constraints**: Foreign keys, Unique, Not null
- **Transactions**: ACID compliant
- **Indexing**: Automatic on primary/foreign keys
